Uma calculadora simples implementada em Go que permite ao usuário realizar operações básicas como adição, subtração, multiplicação e divisão.

## Descrição

Esta é uma calculadora simples escrita na linguagem de programação Go. O programa solicita ao usuário dois números e a operação desejada, e retorna o resultado da operação.

## Tecnologias Utilizadas

- [Go](https://golang.org/) - A linguagem de programação utilizada para desenvolver a calculadora.

## Funcionalidades

- Realiza operações de adição, subtração, multiplicação e divisão.
- Trata erros, como divisão por zero e operações inválidas.
